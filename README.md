# valet-shell

### install
```
brew tap valet-org/homebrew-valet
brew install valet
valet -v
```

### uninstall
```
brew uninstall --force valet
```

## usage

```
git clone git@github.com:playframework/play-scala-starter-example.git
cd play-scala-starter-example/
valet init
valet start
sbt run

http://localhost:9000/posts
```
